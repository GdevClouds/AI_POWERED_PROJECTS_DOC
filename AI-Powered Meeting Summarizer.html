AI-Powered Meeting Summarizer
Documentation
1. Overview
The Meeting Summarizer is an AI-powered system designed to automatically convert meeting
recordings into structured, actionable insights.
It processes audio files and generates:
● Clear summaries of discussions
● Action items with ownership
● Key entities such as people, dates, and organizations
The system is designed to run locally, ensuring privacy and full control over data.
2. Objectives
● Reduce manual effort in documenting meetings
● Improve clarity and consistency in meeting notes
● Enable faster decision-making through structured outputs
● Provide reusable data for integrations and reporting
3. Key Features
●
🎙
Audio Transcription
Converts meeting recordings into text
●
🧠
AI-Based Summarization
Generates concise and meaningful summaries
●
✅
Action Item Extraction
Identifies tasks, owners, and deadlines
●
🏷
Entity Recognition
Detects names, dates, and organizations
●
📄
Structured Outputs
Provides both human-readable and machine-readable formats
4. Tech Stack
The system is built using a combination of modern AI and backend technologies:
● Programming Language: Python
● Speech-to-Text: OpenAI Whisper
● AI Summarization Model: Llama2 (via Ollama)
● NLP Processing: spaCy
● Environment Management: Python Virtual Environment (venv)
● Optional Acceleration: GPU (CUDA / ROCm support)
Architecture Approach
● Fully local processing (no cloud dependency)
● Modular pipeline (transcription → summarization → extraction → output)
● Designed for easy future integrations
5. Current Status
This project is currently under development.

4. Output is saved in structured formats
5. User reviews and utilizes the results
